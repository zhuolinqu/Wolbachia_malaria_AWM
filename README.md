## Overview

This work presents a mathematical modeling framework to assess the potential of *Wolbachia*-based interventions in reducing malaria transmission. The model captures both mosquito population dynamics and malaria transmission processes, with simulation studies informed by biologically relevant parameters.

The repository includes code to reproduce all key simulation results and figures presented in the paper.
